Introduction

I want to create ascii art table from writer or spreadsheet table. I only found one really usable solution called [Senseful Solutions].

Another great solution, to draw ascii diagrams, is [Ascii flow].

The problem

Writing ascii table is very boring.

Senseful Solutions to the rescue.

The solution

Senseful Solutions is a web service. Just copy'n paste your table content from writer or spreadsheet to the input box and format it.

Sometimes, the copy'n paste from writer to Senseful is bad. In this case, copy'n paste to a spreadsheet, then copy'n pas from the spreadsheet to the Senseful web service.

In a matter of milliseconds, you have an awesome ascii table.
